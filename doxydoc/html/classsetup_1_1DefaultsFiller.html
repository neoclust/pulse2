Detailed Description

Replaces declared occurences in a templated file.
Based on a pattern of defaults.

Format of pattern defaults file:
PARAM1 = value1
PARAM2 = value2
...

Format of templated config file with parametters to fill:
[section]
param1 = @@PARAM1@@
param2 = @@PARAM2@@
