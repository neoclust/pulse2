75  Example: On table Software, you can search on ProductName and ProductVersion
76 
77  @param table: table who will be requested
78  @type table: str
79 
80  @param cols: the 2 fields of table who will be requested, it's a string separated by ":"
81  @type cols: str

83  @param value1: ajax search on field 1 (the first field of table)
84  @type value1: str
85 
86  @param value2: ajax search on field 2 (the second field of table)
87  @type value2: str

90  if value1 != '':

93  else: # Search of possibles values of field 2 according to field 1
94  return getValueFuzzyWhere(table, cols.split(':')[0], value1, cols.split(':')[1], value2)
95  return getListValue
